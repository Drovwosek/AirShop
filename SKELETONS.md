# Скелетоны для состояний загрузки

## Что было добавлено

### 1. Базовый компонент Skeleton
**Файл:** `components/ui/skeleton.tsx`

Универсальный компонент скелетона с анимацией shimmer (мерцающим эффектом), который можно использовать для создания любых состояний загрузки.

### 2. Специализированные компоненты скелетонов
**Файл:** `components/skeletons.tsx`

Созданы следующие компоненты для разных частей приложения:

#### `ProductCardSkeleton`
- Скелетон для карточки товара в основной сетке
- Повторяет структуру реальной карточки (изображение, заголовок, цена, кнопка)

#### `CategoriesSkeleton`
- Скелетон для кнопок категорий
- Показывается во время загрузки списка категорий

#### `DiscountSectionSkeleton`
- Скелетон для секции товаров со скидками
- Включает карусель из 3 карточек с полной структурой

#### `SearchResultSkeleton`
- Скелетон для результатов поиска в выпадающем списке
- Компактный формат с миниатюрой и текстом

#### `CartItemSkeleton`
- Скелетон для товаров в корзине (для будущего использования)

#### `ProductDetailSkeleton`
- Скелетон для модального окна детальной информации о товаре (для будущего использования)

#### `PageSkeleton`
- Скелетон для полной страницы
- Включает шапку, категории, поиск, секцию скидок и сетку товаров
- Показывается при первой загрузке приложения

## Где применены скелетоны

### 0. Первая загрузка страницы
При первом открытии приложения показывается полноценный скелетон всей страницы вместо текста "Загрузка...".

### 1. Категории товаров
При первой загрузке страницы вместо пустого места показываются скелетоны кнопок категорий.

### 2. Поиск товаров
Во время ввода текста в поиск и ожидания результатов показываются скелетоны результатов поиска (3 штуки).

### 3. Секция скидок
При первой загрузке страницы показывается скелетон секции со скидками с 3 карточками.

### 4. Сетка товаров
Основная сетка товаров показывает скелетоны карточек во время загрузки (количество зависит от выбранного лимита: 10, 24 или 48).

## Особенности анимации

Используется плавная анимация **shimmer** - эффект мерцающей волны, которая проходит по скелетону слева направо. Это создаёт более живое и приятное восприятие состояния загрузки.

Анимация настроена в:
- `components/ui/skeleton.tsx` - применение анимации
- `app/globals.css` - определение keyframes для анимации

## Адаптивность

Все скелетоны полностью адаптивны и соответствуют реальным компонентам на всех размерах экрана:
- Мобильные устройства (< 640px)
- Планшеты (640px - 1024px)
- Десктоп (> 1024px)

## Рекомендации по использованию

1. **Всегда показывайте скелетоны** вместо пустого экрана или текста "Загрузка..."
2. **Количество скелетонов** должно соответствовать ожидаемому количеству элементов
3. **Структура скелетона** должна максимально точно повторять структуру реального компонента
4. **Длительность анимации** 2 секунды обеспечивает плавность без раздражения

## Как добавить новый скелетон

1. Создайте компонент в `components/skeletons.tsx`
2. Используйте базовый компонент `Skeleton` из `components/ui/skeleton.tsx`
3. Повторите структуру целевого компонента
4. Примените скелетон в нужном месте с проверкой состояния загрузки

Пример:
```tsx
{loading ? (
  <MyComponentSkeleton />
) : (
  <MyComponent data={data} />
)}
```
